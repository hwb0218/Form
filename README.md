## currying

- 여러개의 인자를 받는 함수를 단일 인자를 받는 함수로 체이닝하는 기법
- 동일한 인수를 반복적으로 사용할 때 인수를 고정함으로서 중복을 최소화할 수 있다.
- 커링은 인자의 순서가 중요하므로, 변동 가능성이 적은 인자는 앞에 변동 가능성이 큰 인자는 뒤에 배치한다.
